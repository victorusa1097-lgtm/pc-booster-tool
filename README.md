# PC Toolkit Pro

PC Toolkit Pro is a lightweight Windows optimization tool designed to improve system performance quickly.

## Features
- Speed Booster
- Internet Optimization
- Gaming Mode
- Full System Optimization
- Temporary File Cleaner

## How to Use
1. Download the latest version from the **Releases** section.
2. Extract the ZIP file.
3. Run `PC-Toolkit-Pro.exe` as Administrator.

## Requirements
- Windows 10 / Windows 11
- Administrator privileges

## Disclaimer
This tool performs system optimizations by running Windows commands. Use at your own discretion.
